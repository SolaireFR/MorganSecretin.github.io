SMM: Safe Money Manager

C'est un de mes projets personnels les plus ambitieux. J'ai pour objectif de créé un gestionnaire de compte. J'y ai passer beaucoup d'heure en python pour arriver à un résultat bien avancer. Seuleument, j'ai beaucoup trop appris sur la programmation ce qui ma obligé à recommencer ce code plusieurs fois. Au final, j'ai décidé de le passer dans une version java avec la bibliothéque JavaFX. Cela me permet d'apprendre par moi même dans un logiciel qui un jour pourrai vraiment servir. La version java est toujours en cours sur GitHub.

PCDD
